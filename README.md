# 🔬 Scientific Calculator in Java

A feature-rich console-based Scientific Calculator developed in Java. This project provides basic arithmetic operations, logarithmic functions, trigonometric calculations, roots, factorials, percentage calculations, and many other mathematical utilities through a simple menu-driven interface.

---

## 📌 Features

### Basic Operations
- Addition
- Subtraction
- Multiplication
- Division
- Modulo
- Power (x^y)

### Roots & Logarithms
- Square Root
- Cube Root
- Nth Root
- Logarithm (Base 10)
- Natural Log (ln)
- Logarithm with Custom Base

### Trigonometric Functions
- sin(x)
- cos(x)
- tan(x)
- asin(x)
- acos(x)
- atan(x)

### Hyperbolic Functions
- sinh(x)
- cosh(x)
- tanh(x)

### Constants
- π (Pi)
- e (Euler's Number)

### Utility Functions
- Factorial
- Absolute Value
- Ceiling
- Floor
- Round
- Reciprocal (1/x)
- Percentage Calculation

### Additional Features
- Degree/Radian Mode Toggle
- Error Handling for Invalid Inputs
- Division-by-Zero Protection
- User-Friendly Menu Interface

- Java Math Library (`java.lang.Math`)

## ▶️ How to Run

### 1. Clone the Repository

```bash
git clone https://github.com/abhisekyadav9116300/Scientific-calculator-by-Abhisek
```

### 2. Navigate to the Project Directory

```bash
cd scientific-calculator-java
```

### 3. Compile the Program

```bash
javac ScientificCalculator.java
```

### 4. Run the Program

```bash
java ScientificCalculator
```
